The Oracle

The Oracle is an all-wise deity in the form of a statue placed at the end of a bridge. Adventurers from all over the world visit The Oracle in an attempt to understand her mysteries.

Setting

An old stone bridge high above a gorge enshrouded in dead oak tree branches and darkness with a full moon in the sky and owls that can be heard in the distance. In the center of the bridge is a giant, wooden door that is closed and locked.

Game Information

This game is called "Oracle's Gate". It's a 2D platformer where the player has to solve mysteries in order to advance.

The player can exit the game by closing the browser tab.

The player can start over by refreshing the page.

The player can move by using A and D or the left and right arrow keys.

There is no way to jump in this game.

There is no combat in this game.

Game Lore

The player is the adventurer.

The Oracle is not an adventurer.

The adventurer is on the bridge with The Oracle.

The door's password is "BLACK CAT".

The adventurer can open the door by standing in front of it and saying the door's password.

Nobody knows what's beyond the door.
